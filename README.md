# facef-pos-pratica02
Projeto da atividade 02 da aula de Fundamentos das Técnicas e Práticas de Desenvolvimento - Pós graduação

● TDD - Prática de Testes Unitários e Mocks;

● Containers


## Setup
* java 11
* maven

## Execução
`mvn compile`

## Testes
`mvn test`

## Geração jar
`mvn clean package`

## Gerar imagem
`docker build -t nome:version .`
